# Formulario Clase

Proyecto de práctica realizado para aprender el flujo de trabajo colaborativo utilizando **Git y GitHub**.

El proyecto consiste en un formulario web sencillo desarrollado con:

- HTML
- CSS
- JavaScript

---

## Objetivo

Aplicar conceptos básicos de control de versiones y trabajo colaborativo:

- Crear un repositorio.
- Clonar un proyecto.
- Crear ramas.
- Realizar cambios en el código.
- Crear commits.
- Subir cambios a GitHub.
- Abrir Pull Requests.
- Revisar código.
- Fusionar cambios (Merge).

## Cómo ejecutar el proyecto

1. Clonar el repositorio:

```bash
git clone URL_DEL_REPOSITORIO
```

2. Abrir la carpeta:

```bash
cd Formulario-clase
```

3. Ejecutar:

Abrir `Index.html` en el navegador.

---

## Flujo de trabajo utilizado

### Crear una rama

```bash
git checkout -b nombre-rama
```

### Verificar cambios

```bash
git status
```

### Preparar archivos

```bash
git add .
```

### Crear commit

```bash
git commit -m "Descripción del cambio"
```

### Subir rama

```bash
git push origin nombre-rama
```

### Crear Pull Request

Desde GitHub:

```text
Pull requests
↓
New Pull Request
↓
Review
↓
Merge
```
